Headlinie API
================

API for Headlinie

API
================

```/``` - All sources

```/sources``` - All sources

```/sources/{:name}``` - Specific source

```/sources/{:name}/articles``` - All articles from source

```/sources/{:name}/{:article_url}``` - Article data from encodeURIComponent with article url

Run
================

* Clone repository

* Run ```scripts/bootstrap.sh --dev```
